# 项目简介

## 核心功能
- 基于 YOLOv8 的航拍道路人车检测系统
- 支持对图片、视频及摄像头输入进行检测
- 提供 6 类目标检测：自行车、公交车、汽车、摩托车、人员、卡车
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和 IOU 阈值
- 提供模型选择功能
- 支持批量检测并查看所有结果
- 支持检测结果保存

## 适用场景
- 航拍道路监控与交通分析
- 智能交通管理系统
- 车流与人流统计
- 道路安全监测
- 城市规划数据分析

## 技术特点
- 采用 Python 3.9 开发，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等库
- 包含 2000 张标注数据集
- 提供完整训练好的模型及训练结果
- 支持 PyCharm+Anaconda 或 VSCode+Anaconda 开发环境
- 包含完整的程序文件、UI界面源文件、数据集图片和运行教程
